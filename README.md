# Online courses

Проект с карточками.
Демо: https://online-courses-orcin.vercel.app/

## Функционал

- отображение карточек курсов
- поиск по автору и заголовку
- фильтр по тегу (категории курса)
- адаптивная верстка
- подгрузка новых карточек в DOM-дерево по нажатию на кнопку "Load more" внизу

## Технологический стек

- HTML
- Vanilla JS
- SCSS
- использована [БЭМ-методология](https://ru.bem.info/methodology/)
